# Task Management API

A simple RESTful API for managing tasks with authentication.

## Features

- User Registration & Login
- JWT Authentication
- Role-based Access Control
- CRUD operations for Tasks
- Input Validation using express-validator
- Global Error Handling

## Tech Stack

- Node.js
- Express.js
- MongoDB & Mongoose
- bcrypt
- JWT
